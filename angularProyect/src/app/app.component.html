Directiva ngSwitch

La directiva ngSwitch en Angular permite mostrar o esconder elementos basándose en una expresión. Funciona de manera similar a una declaración switch en otros lenguajes de programación. Se utiliza junto con ngSwitchCase y ngSwitchDefault para definir las diferentes condiciones y casos.

Ejemplo:

El color es el Azul

El color es el Verde

El color es el Rojo

No hay ningún color

Directiva ngStyle

La directiva ngStyle en Angular permite aplicar estilos CSS de manera dinámica a un elemento HTML. Se utiliza para establecer uno o más estilos CSS en un elemento en función de una expresión. Esto es útil para cambiar estilos en tiempo de ejecución basándose en datos del componente.
